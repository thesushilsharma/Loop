# Loop - Find Your Future

Explore universities, read student reviews, and join community discussions to make the right choice.

## Features

- User Account and Profile
- Newsletter
- Discussions, Polls
- University Reviews
- Individual university pages - Detailed views with reviews
- Community forums - Discussion threads

## Usage

## Prerequisites

- VS Code or any Code Editor
- Node.js installed on your machine
- npm package manager

## Technology Stack

- Next.js
- Neon (PostgreSQL) with Drizzle Orm
- Kinde
- Typescript
- Tailwind CSS
- Framer Motion
