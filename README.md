# Loop

Bring Your Own Laptop - Building Blocks for Students

## Features

- User Account and Profile
- Newsletter
- Discussions, Polls

## Usage

## Prerequisites

- VS Code or any Code Editor
- Node.js installed on your machine
- npm package manager

## Technology Stack

- Next.js
- Neon (PostgreSQL) with Drizzle Orm
- Kinde
- Typescript
- Tailwind CSS
